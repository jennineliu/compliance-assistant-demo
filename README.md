# compliance-assistant-demo
An interactive decision support tool that transforms complex international customs regulations into clear, traveler-friendly compliance guidance.

## 🚀 Live Demo
[Launch Compliance Assistant](https://jennineliu.github.io/compliance-assistant-demo/)

## Overview

Compliance Assistant helps travelers understand whether items in their luggage are:

- ✅ Allowed
- ⚠️ Require declaration
- 🚫 Prohibited

Instead of reading lengthy government regulations, travelers receive clear recommendations, estimated duties, declaration requirements, and links to official government resources.

The current implementation focuses on Brazil and serves as a reusable framework for future country-specific compliance assistants.

---

## Features

- Interactive searchable customs registry
- Risk-based item classification
- Customs status indicators
- Import duty calculator
- Arrival-specific duty exemptions
- Currency declaration guidance
- Drone compliance information
- Medication guidance
- Food import restrictions
- Packing checklist
- Mobile-first responsive design
- PDF export
- Official government references

---

## Why I Built This

As an intelligence analyst, I wanted to explore how AI-era design principles can simplify highly regulated information.

Government regulations are often accurate but difficult to navigate. This project demonstrates how complex policy can be transformed into intuitive decision support through thoughtful information architecture and user experience design.

Rather than replacing official guidance, the application helps travelers quickly determine:

- What they can bring
- What requires declaration
- What may incur duties
- Where to find authoritative information
